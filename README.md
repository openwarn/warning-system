# Warning-Director

Orchestrierung der Services des Warnsystems.
Dieses Projekt ist Teil der Masterarbeit von Alexander Noack (TU Ilmenau; Fachgebiet Softwarearchitekturen und Produktlinien).

## Voraussetzungen

* Docker >= v18.06
* Docker-Compose >= v1.23.2

## System starten

```bash
docker-compose up -d
```
